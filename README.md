# Clean Code

## Install:

> - git clone "repo"
> - cd"repo"
> - npm i

## Git:

> - git fetch
> - git pull

> - git branch "feature/name"
> - git checkout "feature/name"

> - git add .
> - git commit -m "commit..."
> - git push

## Firebase deploy:

> - npm run build
> - firebase deploy
